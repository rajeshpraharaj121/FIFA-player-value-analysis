# ⚽ FIFA Player Value Analysis

## 📌 Project Overview
This project analyzes football player market values using real-world data.  
The goal is to uncover insights about how **age, nationality, and league** affect player value.

---

## 📊 Dataset
- Source: Transfermarkt dataset
- Rows: 500+
- Columns: 30+

---

## 🛠️ Tech Stack
- Python
- Pandas
- Matplotlib

---

## 🔍 Key Analysis

### 1. Age vs Player Value
- Player value peaks around ages **24–27**
- Declines gradually after peak age

### 2. League Analysis
- Premier League has highest average player value
- Indicates strong financial power and talent density

### 3. Nationality Analysis
- Countries like Spain, France, England produce most players
- Some countries have higher average value due to elite players

---

## 📈 Visualizations
- Line chart (Age vs Value)
- Bar chart (Top countries)
- League comparison charts

---

## 🧠 Key Insights
- Prime performance age = mid 20s
- League impacts market value significantly
- Player distribution varies by country

---

## 🚀 Conclusion
This project demonstrates how data analysis can uncover patterns in sports economics and player valuation.
